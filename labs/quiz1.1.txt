1. A tilde represents the home directory of the current user.
2. An apsolute path is the full path through the directories to your current location. An absolute path is start with a / and the first directory created.
3. A relative path starts from your current directory.
4. ls Download
5. ls -R test
6. ls /Desktop
7. Each utility solves one particular problem.  that way the utilities can be used with each other. 
8. pwd
9. How do you get your previous command to appear on the command line without retyping it? 
	Up arrow key	
Using a relative path what would the command be to show the contents of a directory one level higher than your current location? 
	ls [directory]
Using an absolute path (not using the tilde shortcut) give the command to open a file in your home directory called SampleFile.txt with the vi editor.  
	vi /home/SampleFile.txt	
What is the command to display the manual page for the less paging utility? 
	man less

	/input
